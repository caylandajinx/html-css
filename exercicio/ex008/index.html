Princípais formatações

Negrito / Destaque

Nesta frase, temos u termo em negrito usando a tag <b> (não semântica).

Nesta frase, tenho um termo em destaque usando a tag <strong> (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não semântica).

Nesta frase, temos um termo em ênfase usando a tag <em> (semântica).

Marca texto

Podemos criar também um texto marcado usando a tag <mark>.

E no outro parágrafo, temos outro texto marcado no final.

Texto "grande" e texto "pequeno"

Estamos criando um texto grande com a tag <big>(obsoleto) e um texto pequeno com a tag <small> nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluído usando a tag <del> para indicar que ele deve ser lido, mas não considerado (semântica).

Texto inserido

Podemos marcar um texto como inserido usando a tag <ins> para dar uma ênfase e indicar que ele foi adicionado depois (semântica).

Existe também o sublinhado com tag <u> (não semântica).

Texto sobrescrito

Para inserir coisas do tipo x20+3 com a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O com a tag <sub>
