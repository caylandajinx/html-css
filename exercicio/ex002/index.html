Parágrafos e quebras de linha

Você pode escrever um parágrafo de quarquer jeito. Basta colocar tudo no meio do par de tags <p>e </p>

Se precisar quebrar o texto em algum lugar específico
como esse, você pode usar a tag <br>.

Vamos adicionar alguns símbolos:

  •   Espaço em branco
  • ® / Marca registrada
  • © / Copyright
  • ™ / Trade Mark
  • € / Euro
  • £ / Libra (pound)
  • ¥ / Yen
  • ¢ / Cent
  • ∅ / Vazio
  • ∑ / Soma
  • δ / Delta
  • ← / Seta esquerda
  • ↑ / Seta acima
  • → / Seta direita
  • ↓ / Seta baixo

Vamos adicionar alguns emoji:

  • 🖖
  • 🤓
  • ✨
