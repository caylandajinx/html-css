Uma lista ordenada, utiliza a tag <ol> que delimita a lista <li> que identifica cada item. (a tag <li> tem o fechamento opcional)

A tag <ol> possui um parâmetro type= onde pode mudar o marcado da lista atual. As opções dos parâmetro são:
⤑ 1 = ex:1, 2, 3, 4, ...
⤑ A = ex:A, B, C, D, ...
⤑ a = ex:a, b, c, d, ...
⤑ I = ex:I, II, III, IV, ...
⤑ i = ex:i, ii, iii, iv, ...

Também pode indicar o início da contagem usando o parâmetro start.
Por exemplo, a tag <ol type=Istart = 5> vai gerar itens numerados como V, VI,...

Para criar uma unordered list, também chamadas de listas com marcadores. Utilizando a tag <ul> junto com <li> para criar os itens.

Podemos configurar a propriedade type da tag <ul type=circle> com os seguites valores:
⤑ disc = padrão. Uma bolinha totalmente preechida
⤑ circle = Uma bola com borda preta e sem preenchido
⤑ square = Um pequeno quadrado totalmente preenchido

Listas mistas

Podemos configurar listas dentro de outras listas. Além de juntar listas <ul> com <ol> e vice-versa. As listas internas sempre terão deslocamento interno para a direita.

Lista de Definições

É como um dicionário, temos os termos e suas descrições.

As listas de definições tem a tag <dl>
Os termos tem a tag <dt>
As descrições tem a tag <dd>.
