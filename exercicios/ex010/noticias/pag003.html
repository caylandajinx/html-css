Para poder controlar onde o site de destino vai abrir, podemos usar o atributo target, que suporta os seguintes valores:

  • _blank
  • _self
  • _top
  • _parent

Atributo rel= . Esse atributo aceita vários valores:

  • next
  • prev
  • author
  • external

Para voltar a pasta atual se usa ./ e para voltar a pasta anterior se usa ../
