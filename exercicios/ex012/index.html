Formatos

  3. OGG

É possível utilizar o software handbreake para a convesão dos arquivos de vídeo e outros online para o OGG.

Dentro da tag <video> se utiliza a tag <source> com o atributo src= e type=

Na tag <video> configuramos alguns atributos importantes:

  • width vai indicar a largura que o vídeo vai ter na tela. Exemplo 600px
  • post Configura uma imagem que vai aparecer como capa, enquanto não aperta o play.
  • controls Configura se os controles do vídeo vão aparecer na parte inferior da mídia.
  • autoplay Diz para o navegador se o vídeo vai começar a tocar automaticamente, assim que a página for carregada.
