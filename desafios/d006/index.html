Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deverá utilizar em cada caso.

A <;!DOCTYPE html> deve aparecer uma vez, no top da página (antes de qualquer tag HTML).

  1. Títulos HTML
    • <h1 ao h6>
  2. Parágrafo, etc..
    • <Lorem>
    • <p>
    • <br>
    • <hr>
  3. Trechos de códigos
    • <pre>
    • <code>
  4. Elementos de formatação HTML
    • <b>
    • <strong>
    • <i>
    • <em>
    • <mark>
    • <small>
    • <del>
    • <ins>
    • <sub>
    • <sup>
  5. Elementos de Citação e abreviação
    • <q>
    • <blockquote>
    • <cite>
    • <abbr>
    • <address>
    • <bdo>
  6. Comentários
    • <!-->
